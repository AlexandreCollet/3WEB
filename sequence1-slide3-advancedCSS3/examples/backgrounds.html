Background Origin

div { background-origin: padding-box|border-box|content-box; }

Content box

div { background-origin: content-box; }

Padding box (default)

div { background-origin: padding-box; }

Border box

div { background-origin: border-box; }

Linear gradiant

div { background: linear-gradient(direction, color-stop1, color-stop2, ...); }

Classic

div { background: linear-gradient(red,yellow,green); }

Direction

div { background: linear-gradient(45deg,red,yellow,green); }

Differently Spaced

div { background: linear-gradient(red,yellow 20%,green 90%); }

Radial gradiant

div { background: radial-gradient(shape size at position, start-color, ..., last-color); }

Classic

div { background: radial-gradient(red,yellow,green); }

Shape

div { background: radial-gradient(circle,red,yellow,green); }

Repeating

div { background: repeating-radial-gradient(red, yellow 10%, green 15%); }

Box reflect

div { -webkit-box-reflect: direction margin mask-box; }

Below

div { -webkit-box-reflect: below 5px linear-gradient(transparent, white); }
